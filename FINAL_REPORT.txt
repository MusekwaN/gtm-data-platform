## 📋 How Each Script Works

### 1. BRONZE_INGESTION.PY
```
INPUT:  Kafka topics (apollo_leads, crm_events)
PROCESS: Raw ingestion, add metadata
OUTPUT: ./data/bronze/leads/ (Parquet)
TRIGGER: Every 30 seconds
COLUMNS: 18 (raw + metadata)
```

**Expected Output:**
```
Starting Bronze ingestion layer...
Bronze streams running. Waiting for data...
Batch [0] written 23 records to ./data/bronze/leads
Batch [1] written 45 records to ./data/bronze/leads
...
```

### 2. SILVER_PROCESSING.PY
```
INPUT:  ./data/bronze/leads/
PROCESS: Clean, deduplicate, standardize
OUTPUT: ./data/silver/leads/ (Parquet)
TRIGGER: Every 60 seconds
COLUMNS: 14 (cleaned + consolidated)
```

**Expected Output:**
```
Starting Silver processing layer...
Silver stream running...
Batch [1] processed 23 records (cleaned & deduplicated)
✓ Emails normalized (lowercase)
✓ Email domains extracted
✓ Full names concatenated
✓ 3 duplicate records removed
...
```

### 3. GOLD_LEAD_SCORING.PY
```
INPUT:  ./data/silver/leads/
PROCESS: Score leads, classify intent
OUTPUT: ./data/gold/lead_scores/ (Parquet)
MODE: Batch processing (one-time)
COLUMNS: 16 (scores + actions)
```

**Expected Output:**
```
Running Gold lead scoring...
Reading 126 records from ./data/silver/leads

Scoring Applied:
✓ Seniority scores calculated (5-40 points)
✓ Company size scores calculated (5-30 points)
✓ Industry fit scores calculated (5-20 points)
✓ Data completeness scores calculated (0-10 pts)

Gold lead scores written successfully
```

## 🚀 How to Run

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Ensure Kafka is Running
```bash
# Verify Kafka broker on localhost:9092
# Create topics: apollo_leads, crm_events
```

### Step 3: Run Scripts (3 Terminals)

**Terminal 1:**
```bash
python streaming\spark_jobs\bronze_ingestion.py
```
Wait for: "Bronze streams running..."

**Terminal 2:** (after Bronze starts)
```bash
python streaming\spark_jobs\silver_processing.py
```
Wait for: Data appears from Bronze

**Terminal 3:** (after Silver has data)
```bash
python streaming\spark_jobs\gold_lead_scoring.py
```
Should complete and show results
